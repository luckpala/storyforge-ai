# StoryForge AI - 版本说明

本项目包含两个独立版本：

## 1. React 版本（当前目录）
- **位置**：`D:\storyforge-ai`
- **技术栈**：React + TypeScript + Vite
- **启动方式**：
  - `启动StoryForge.bat` - 开发模式
  - `启动应用.bat` - 生产模式（需要先运行 `npm run build`）
  - `启动应用和代理.bat` - 启动应用和代理服务器

## 2. Python 版本（独立目录）
- **位置**：`D:\storyforge-python`
- **技术栈**：Python + Streamlit
- **启动方式**：
  - `快速启动.bat`
  - `启动完整版.bat`
  - `启动应用.bat`

## 注意事项
- 两个版本完全独立，可以同时运行
- React版本是主要开发版本，功能更完整
- Python版本是实验性版本，功能较简单
